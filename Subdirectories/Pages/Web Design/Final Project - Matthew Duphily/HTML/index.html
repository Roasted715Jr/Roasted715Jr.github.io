Network and Computer Systems Administrator

This job is just how it sounds: you will administer a network and the within computer system(s) to ensure they are secure and working properly. You need to know how to connect computers to a network, and get that network up and running for as long as you need it to. Networks are what connect us to the internet, so this occupation can be very useful for everyone: the administrator and the administrator's clients.
