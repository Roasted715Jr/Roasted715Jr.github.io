What will I be doing in this occupation?

As a network and computer systems administrator, you may do these tasks:

  • Install, configure, and support an organization's local area network (LAN), wide area network (WAN), and Internet systems or a segment of a network system
  • Monitor network to ensure network availability to all system users and may perform necessary maintenance to support network availability
  • May monitor and test Web site performance to ensure Web sites operate correctly and without interruption
  • May assist in network modeling, analysis, planning, and coordination between network and data communications hardware and software
  • May supervise computer user support specialists and computer network support specialists
  • May administer network security measures

As a network and computer admininistrator, you earn about $37.41 hourly and $77,810 annual (median for 2015).

Computer science is used in this occupation because you have to use problem solving. You need to find the shortest and easiest way to network all the computers that will be connected to the network. Solving this problem can involve the use of a minimal spanning tree.
